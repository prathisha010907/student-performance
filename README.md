##  Overview
This project focuses on analyzing and predicting student performance using machine learning techniques. It helps in identifying patterns in student data, predicting outcomes, and grouping students based on their performance.

The goal of this project is to make performance analysis faster, more accurate, and useful for early intervention.

---

##  Features
-  Data Analysis and Visualization
-  Multiple Machine Learning Models
-  Performance Prediction (Regression)
-  Classification (Pass/Fail & Grades)
-  Student Clustering using KMeans
-  Model Accuracy Comparison

---

## 📂 Dataset
- Source: Kaggle Student Performance Dataset
- Total Records: 1000 students
- Features:
  - Gender
  - Parental Education
  - Test Preparation
  - Math Score
  - Reading Score
  - Writing Score

---

## 🛠 Technologies Used
- Python
- Pandas (Data Handling)
- NumPy (Numerical Computation)
- Scikit-learn (Machine Learning Models)
- Matplotlib & Seaborn (Data Visualization)
- Google Colab (Development Environment)

---

## 🤖 Machine Learning Models Used
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest ⭐ (Best Performing)
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Naive Bayes
- KMeans Clustering (Unsupervised Learning)

---

## 📊 Results
- Random Forest achieved the highest accuracy (~92%)
- Students who completed test preparation performed better
- Strong correlation between reading and writing scores
- Students were grouped into 3 categories:
  - High Performers
  - Average Performers
  - Low Performers
